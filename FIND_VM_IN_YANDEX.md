# Как найти VM в Yandex Cloud

## Способ 1: Через список виртуальных машин

### Шаг 1: Откройте консоль Yandex Cloud

1. Перейдите: https://console.yandex.cloud
   - Или для Казахстана: https://kz.console.yandex.cloud

2. **Войдите** с аккаунтом заказчика

### Шаг 2: Выберите правильный каталог

1. В верхней части страницы найдите **выбор каталога** (dropdown)
2. Выберите каталог: `ao7gso7afqnk7771b5l1` (из URL который дал заказчик)
   - Или найдите каталог по названию

### Шаг 3: Перейдите к виртуальным машинам

1. В левом меню найдите **Compute Cloud**
2. Нажмите на **Compute Cloud** → откроется подменю
3. Нажмите **Виртуальные машины**

### Шаг 4: Найдите вашу VM

В списке виртуальных машин найдите:
- **По IP адресу:** `94.131.80.213`
- **По имени:** может быть `kyte-backend` или другое имя
- **По статусу:** должна быть **Running** (Запущена)

---

## Способ 2: Поиск по IP адресу

1. В консоли Yandex Cloud используйте **поиск** (вверху страницы)
2. Введите IP: `94.131.80.213`
3. Нажмите Enter
4. Должна найтись ваша VM

---

## Способ 3: Через прямой URL

Если знаете ID VM, можно открыть напрямую:

```
https://console.yandex.cloud/folders/ao7gso7afqnk7771b5l1/compute/vm/<vm-id>
```

Но проще использовать способ 1.

---

## Что делать после того как нашли VM:

### 1. Откройте VM

Нажмите на название VM в списке

### 2. Найдите раздел "Доступ" или "SSH ключи"

В открывшейся странице VM найдите:
- **Вкладка "Доступ"** или
- **Раздел "SSH ключи"** или
- **"Подключение"**

### 3. Добавьте SSH ключ

1. Нажмите **"Добавить ключ"** или **"Изменить"**
2. Вставьте публичный ключ:
   ```
   ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIFX4hpHP4ruknY+jjnhhDRUxN2SU2AyQCchlV+O1Q9sN kyte-backend-yandex-cloud
   ```
3. Сохраните

### 4. Проверьте группу безопасности

1. В разделе **Сеть** или **Группы безопасности**
2. Убедитесь что открыты порты:
   - **22** (SSH)
   - **80** (HTTP)
   - **443** (HTTPS)

## Если не можете найти:

1. **Проверьте каталог** - убедитесь что выбрали правильный каталог
2. **Проверьте регион** - возможно VM в другом регионе
3. **Попросите заказчика** показать скриншот или дать прямой URL
4. **Используйте поиск** по IP адресу `94.131.80.213`

1. https://console.yandex.cloud
