# Добавление SSH ключа в Yandex Cloud VM

## Шаг 1: Получите публичный ключ

После создания ключа командой выше, скопируйте публичный ключ из вывода.

Или вручную:
```powershell
cat C:\Users\1\.ssh\yandex_cloud.pub
```

## Шаг 2: Попросите заказчика добавить ключ

### Вариант A: Через консоль Yandex Cloud

1. Откройте: https://console.yandex.cloud
2. Перейдите: **Compute Cloud → Виртуальные машины**
3. Найдите VM с IP `94.131.80.213`
4. Нажмите на VM
5. В разделе **Доступ** или **SSH ключи**
6. Нажмите **Добавить ключ** или **Изменить**
7. Вставьте ваш публичный ключ
8. Сохраните

### Вариант B: При создании/изменении VM

1. В настройках VM найдите раздел **Доступ**
2. В поле **SSH ключ** добавьте ваш публичный ключ
3. Сохраните изменения

## Шаг 3: Подключитесь

После того как заказчик добавит ключ:

```powershell
ssh -i C:\Users\1\.ssh\yandex_cloud kyte-777@94.131.80.213
```

---

## Альтернатива: Используйте существующий ключ

Если хотите использовать существующий ключ `id_ed25519`:

```powershell
# Покажите публичный ключ
cat C:\Users\1\.ssh\id_ed25519.pub

# Попросите заказчика добавить его

# Подключитесь
ssh -i C:\Users\1\.ssh\id_ed25519 kyte-777@94.131.80.213
```
